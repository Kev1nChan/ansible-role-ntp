ansible ntp 角色
=========

在 Linux 上安装 NTP 服务，如果有多台服务器，将设置第一台服务器为 NTP SERVER。

角色变量
--------------

变量请参考项目根目录下 `defaults` 目录，如需覆盖变量请在 `inventory` 或者 `playbook` 中设置变量

Playbook 示例
----------------

示例请参考项目根目录下 `tests` 目录
